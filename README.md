## Introduction
본 프로젝트는 경험을 공유할 수 있는 플랫폼에 대한 저장소입니다.

## Branches
Git flow 방식으로 개발을 진행합니다.
`master` - 배포용 브런치입니다.
`develop` - 개발용 브런치입니다. 
`features/[x]` - 개발용 브런치기반으로 하여 세부개발 브런치로 x에는 기능이름이 들어갑니다. 
`hotfix` - 배포/운용 중 긴급한 버그를 해결하기 위한 브런치입니다, master에 머지하는 것을 기본으로 .
